# What file permissions should installed personas use?

## Decision

Write persona YAML files at 0o600 and create parent dirs at 0o700 (not 0o644/0o755). Personas are remotely-fetched reviewer instructions; world-readable mode plus world-traversable `~/.config/atcr` dir lets any other user on a shared host read every installed persona. This applies to both `install.go` and `upgrade.go`. Verify with a stat-mode assertion in the install test. Related: use atomic write (temp file + os.Rename) to avoid partial writes from crashes — `internal/atomicfs` is available but `internal/personas` must add the import (TD-010).
