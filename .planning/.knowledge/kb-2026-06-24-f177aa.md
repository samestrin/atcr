# How should corroboration scores be passed into SelectEligibl

## Decision

Use a caller-supplied `map[string]float64` as a 4th argument (reviewer name → corroboration rate). A `nil` map is valid "no score data" — the matched partition falls back to alphabetical-only ordering. Chosen over a functional option (avoids Option-type machinery for a single optional input) and over a new overload (avoids keeping a scoreless path first-class). Only one production caller exists (`internal/verify/pipeline.go`), so the signature change is contained. T6 (`scorecard.Aggregate()` → `LeaderboardRow.CorroborationRate` keyed by reviewer) and T8 (SelectEligibleSkeptics score lookup) share one carrier shape. Key convention: lowercase reviewer name on both sides to avoid silent miss on mixed-case agent names (TD-013).
