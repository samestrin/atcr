# Why must .git directory matching use case-insensitive compar

## Decision

Walker recursion does not route child paths through jail.Resolve. On case-preserving filesystems, a .GIT (uppercase) dir is the real git dir yet a case-sensitive == .git check descends it, exposing .git/config (can embed credentials). Use strings.EqualFold to match the jail semantics.
