# Where does review-dir/manifest/latest management live in the

## Decision

Review-dir, manifest, and latest pointer management lives in `internal/fanout` (not a separate `internal/reviewdir` package). The original requirements lock the internal package set to 9 packages: gitrange, payload, registry, llmclient, fanout, stream, reconcile, report, mcp. Task #7 in the original requirements bundles "review-directory/manifest/latest management" into the fan-out engine. The AC 01-03 "internal/reviewdir/creator.go" file path was superseded by the locked architecture. The `payload.Manifest` type already existed, so fanout was the natural home.
