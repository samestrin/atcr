# Why does single-turn prevSigs repeat detection miss ABAB osc

## Decision

prevSigs only remembers the immediately-prior turn, so an alternating pattern A,B,A,B never matches. Track a bounded ring of K=3-4 turns and trip when any signature reappears. Also record loop-hygiene halts in TrippedBudgets so operators can diagnose early stops.
