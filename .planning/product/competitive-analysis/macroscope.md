# Macroscope

## Overview
Macroscope (macroscope.com) is an AI-powered code review and engineering workflow platform. Its primary pitch is "A macroscope for your engineering organization," focusing on merging PRs faster, automating status updates, and fixing bugs before production.

## Core Features
- **AI Code Review**: Automated PR reviews and bug detection.
- **Automated PR Descriptions & Status**: Automatically summarizes what is changing and generates PR descriptions.
- **Engineering Analytics**: Dashboards tracking metrics like "Pushed vs Landed Coding Time," contributor output, and project focus.
- **Agentic Actions**: An Agent that "answers questions and takes action."

## Relevance to ATCR
Macroscope is a **direct competitor** in the AI Code Review space, sitting alongside CodeRabbit and Codium. 

**What they do well**: They are heavily targeting Engineering Managers and VPs by leading with analytics. Instead of just selling "better code," they sell "visibility into engineering velocity" (dashboards showing who is pushing code, how fast it lands, etc.).

## ATCR Roadmap Alignment
- **Validates Epic 42.0 (Review Intelligence Analytics)**: Macroscope proves that enterprise buyers want dashboards. When we execute Epic 42.0, we need to ensure our analytics provide deep managerial insights (e.g., tech-debt reduction over time, time-to-merge, multi-tier fix success rates).
- **Potential New Feature**: If we don't already have it, we should add **Automated PR Description Generation** to our auto-fix pipeline. Developers love when the AI not only fixes the code but writes the Git PR summary.
