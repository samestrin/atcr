# Epic Plan 51.0: Skeptic v2 - Automated Red Teaming & OWASP Guardrails

- **Estimated time**: 2 weeks
- **Tasks/Components**: TBD
- **Execution**: init-plan

## Objective

Evolve the "Skeptic" persona from a simple "Cross-Examiner" (Epic 6.0) into a formal **Automated Red Teaming** agent. This ensures ATCR provides enterprise-grade adversarial security testing on every PR, specifically guarding against AI-introduced vulnerabilities and Inter-Agent Attacks.

## Context

Mend.io and OWASP have formalized the risks of AI coding agents. A major risk in multi-agent systems is the **Inter-Agent Attack**: if the Auto-Fixer agent generates a patch containing a hidden prompt injection or vulnerable payload, the downstream Skeptic agent might execute it or be compromised by it. We need to harden the orchestration layer between agents and instruct the Skeptic to formally evaluate all generated code against the **OWASP LLM Top 10**.

## Proposed Solution

1. **OWASP Top 10 Evaluation**: Update the Skeptic persona's system prompt (in `personas/community/skeptic.yaml`) to explicitly validate all findings and auto-fixes against the OWASP Top 10 for LLMs (e.g., catching Prompt Injection, Insecure Plugin Design, Training Data Leakage).
2. **Inter-Agent Context Sanitization**: Implement a sanitization layer in the Orchestrator (`internal/personas/runner.go`). When passing the `Fixer`'s output to the `Skeptic` for adversarial verification, strip executable payloads and markdown commands that could hijack the Skeptic's context window.
3. **Dedicated Red Team Persona (Optional)**: Create a hyper-aggressive `RedTeam` persona that specifically tries to craft zero-click exploits or bypasses for the PR code, acting as a continuous penetration tester.

## Acceptance Criteria

- [ ] The Skeptic's prompt formally requires validation against the OWASP LLM Top 10.
- [ ] Context passed from the Fixer to the Skeptic is aggressively sanitized to prevent inter-agent prompt injections.
- [ ] A test suite (`internal/personas/redteam_integration_test.go`) validates that the Skeptic successfully catches deliberately injected OWASP Top 10 vulnerabilities in test PRs.
