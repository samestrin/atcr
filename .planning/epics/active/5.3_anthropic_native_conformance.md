# Epic 5.3: Anthropic-Native Conformance

**Status:** Draft (stub — created 2026-06-19 to track a gap surfaced in epic reconciliation)
**Created:** 2026-06-19
**Priority:** Medium (regression safety for a primary provider)

## Why this exists

Epic 5.1 (OpenAI-Compatible Conformance Suite) deliberately scopes itself to the
`/v1/chat/completions` surface (OpenAI + Ollama) and **explicitly excludes the Anthropic
native API** ("uses a different wire format; separate concern"). Anthropic is a primary
provider for ATCR's reviewer panel, yet no sibling epic picks that concern up — so it was
silently dropped. This stub makes the gap visible and tracked rather than lost.

## Problem

`internal/llmclient` talks to Anthropic's native Messages API, whose wire format differs
from the OpenAI-compatible surface (system prompt handling, `content` blocks, `tool_use` /
`tool_result` blocks, `stop_reason` vs `finish_reason`). 5.1's fixtures will protect the
OpenAI/Ollama path but give **zero coverage** of the Anthropic decode path. A regression in
how ATCR parses an Anthropic `tool_use` response or `stop_reason` would not be caught.

## Goal

Record real wire fixtures from the Anthropic native Messages API and assert the llmclient
parser produces the correct engine-facing `ChatResponse` — the same protection 5.1 gives
the OpenAI-compatible surface, extended to Anthropic.

## Acceptance Criteria (draft — refine before scheduling)

- [ ] AC1: Fixture capture records a live `tool_use` response from the Anthropic Messages API.
- [ ] AC2: Fixture capture records a live plain-text + `stop_reason` response.
- [ ] AC3: Fixtures stored under `internal/llmclient/testdata/conformance/anthropic/`, redacted of any sensitive content.
- [ ] AC4: Table-driven conformance test asserts tool-call argument decoding and `stop_reason` handling produce the correct engine-facing `ChatResponse`.
- [ ] AC5: Test runs offline (static fixtures, no live calls at test time).

## Out of Scope

- The OpenAI-compatible surface (covered by Epic 5.1).
- Load/performance testing.

## Dependencies

- Mirrors Epic 5.1's harness — if 5.1 ships a reusable fixture/test scaffold, 5.3 reuses it.
- Fixture capture requires live Anthropic API access; fixtures must be redacted before commit.
