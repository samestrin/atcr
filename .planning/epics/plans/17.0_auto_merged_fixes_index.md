# Epic 17.0: Auto-Merged Fixes

**Status:** Active
**Created:** 2026-06-25
**Priority:** High (Game-changing capability)
**Target:** Transition ATCR from a passive reviewer to an active contributor.

## Overview

Identifying technical debt is only half the battle. To drastically reduce developer friction, ATCR must be able to automatically apply its suggested fixes, validate them, and merge or propose them via pull requests. This closes the loop on technical debt.
