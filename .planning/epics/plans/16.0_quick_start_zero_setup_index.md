# Epic 16.0: Quick Start & Zero Configuration

**Status:** Active
**Created:** 2026-06-25
**Priority:** High (Crucial for onboarding and time-to-value)
**Target:** Enable users to run their first review within 30 seconds of installation.

## Overview

The biggest drop-off point for any CLI tool is configuration friction. By driving users through a synthetic sign-up flow (with referral tracking) and automatically generating default `.github/workflows` and `.atcr/profiles.yaml`, we can deliver an "Aha!" moment instantly.
