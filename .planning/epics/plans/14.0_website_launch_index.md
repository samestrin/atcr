# Epic Index 14.0: atcr.dev Website Launch

- **Estimated time**: 2 weeks
- **Tasks**: 9
- **Components**: 3

## Context

With Epic 13.0 delivering Reconciler v2 semantic deduplication and grouping, ATCR is mature enough to be showcased publicly. However, recruiting pilots for Epic 15.0 (Team Edition Validation) and onboarding early adopters requires a polished, high-fidelity marketing and documentation home. The selected "The Artifact" design concept matches the product's aesthetic: technical, evidence-forward, paper-like, and highly minimal.

## Problem Statement

1. Personal network recruitment for the Team Edition pilot is insufficient. We need a scaleable, public landing page to capture interest.
2. The engine's reviewer personas and CLI usage patterns are documented in repo markdown files, which are hard for non-contributors to read and digest.
3. The model evaluation leaderboard (Epic 10.0) needs to be publicly visible to establish brand credibility.
4. The website must feel extremely premium, responsive, and follow the site design system (Plex fonts, theme switching, interactive viewer).

## Proposed Solution

Coordinate the rollout of the `atcr.dev` website launch across 4 sprints:
- **[Sprint 14.0: Documentation & Code Review Alignment](../active/14.0_documentation_audit.md)**
- **[Sprint 14.1: Core Landing Page & Design System](../active/14.1_core_landing_page.md)**
- **[Sprint 14.2: Documentation & Reviewer Personas](../active/14.2_docs_and_personas.md)**
- **[Sprint 14.3: Leaderboard & Production Deployment](../active/14.3_leaderboard_and_deployment.md)**

## Acceptance Criteria

- [ ] AC1: All 3 sprints completed and integrated into the `atcr.dev` repository.
- [ ] AC2: Website passes Lighthouse accessibility and SEO audit with scores > 90.
- [ ] AC3: Fully responsive down to 320px mobile viewport (using horizontally scrollable artifact viewports without code wrapping).
- [ ] AC4: Dark mode support via system settings (`prefers-color-scheme`) with no visual flash on load.
- [ ] AC5: Functional command-copy pill, consensus meter, and tabbed artifact viewer with zero external frameworks.

## Success Criteria

### Functional
- Page initial load under 1.5 seconds.
- Interactive artifact tab switching is instantaneous (under 50ms).
- Interactive command pills copy CLI instructions cleanly to clipboard.

### Business
- Establish a reliable signup channel capturing at least 50+ pilot program signups in the first month.
- Increase repository stars and external developer engagement.

## Technical Constraints

- Must be built with pure HTML, vanilla CSS, and vanilla JS (no heavy frameworks like React/Tailwind unless explicitly requested).
- Must be deployed via static hosting (GitHub Pages or Cloudflare Pages).
- Form signups must route securely to a form collector (e.g., Formspree, Netlify Forms, or Google Sheets integration).

## Non-Functional Requirements

### Performance
- Page weight < 500KB total assets.
- Fully offline-testable via relative URLs.

### Accessibility
- Semantic HTML tags used for all layouts (`<nav>`, `<main>`, `<section>`, `<footer>`).
- Contrast ratios pass WCAG AA standards in both light and dark themes.

## Out of Scope

- Hosting a backend database or user account/login system (handled by third-party form handler or future SaaS epics).
- Real-time websocket logs or actual terminal runtime execution on the landing page (uses high-fidelity mockup output files).

## Dependencies

- Repo mockups and design assets from `~/Documents/GitHub/atcr.dev/the-artifact/`.
- Leaderboard evaluations (Epic 10.0) to populate the live standings.
- Roster profiles of reviewers (Epic 9.0) to populate the personas page.

## Risks

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|------------|
| Form integration fails to capture leads | Low | High | Pre-launch test the submit endpoints, setup fallback email alerts, and verify CORS configurations |
| Horizontal code blocks break layouts | Medium | Medium | Force horizontal scrolling container overflow style on all `<pre>` and code blocks |
