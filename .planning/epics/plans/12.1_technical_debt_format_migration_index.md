# Epic 12.1: Technical Debt Format Migration

## Overview
This epic focuses on solving the immediate friction caused by our legacy Markdown table format for tracking technical debt. By migrating to a structured, multi-line friendly format (like individual YAML/Markdown files), we ensure critical context is no longer lost due to formatting constraints.
