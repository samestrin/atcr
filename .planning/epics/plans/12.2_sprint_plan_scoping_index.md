# Epic 12.2: Sprint Plan Scoping in ATCR

## Overview
This epic reintroduces the sprint-plan scoping feature from our legacy system into ATCR. By providing a `--sprint-plan` file path, the code review process can be tightly scoped to the specific tasks and files of the current sprint, greatly reducing noise from unrelated code formatting or minor refactoring changes in the diff.
